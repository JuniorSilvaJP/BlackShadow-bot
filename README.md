# Albion Recruit Bot

PASSO A PASSO

1. Crie o bot:
https://discord.com/developers/applications

2. New Application.

3. Bot -> Add Bot.

4. Ative:
- Server Members Intent
- Message Content Intent

5. Copie o Token.

6. Abra o arquivo .env:
TOKEN=SEU_TOKEN

7. No OAuth2 -> URL Generator:
Scopes:
- bot
- applications.commands

Permissões:
- Manage Roles
- Manage Nicknames
- View Channels
- Send Messages
- Read Message History

8. Convide o bot.

9. No terminal:
npm install
node index.js

Canais:
#recrutamento
#logs-recrutamento

Cargos:
Recruta
Guarda
Líder
Comandante
